## Technology stack

* **Front-end:** [React.Js](https://reactjs.org/)

## Running locally

1. Clone this repo:

```sh
$ git clone https://github.com/iamdevmarcos/minimal-one-page-template.git
```

2. Then go to the project's folder:

```sh
cd macOS
```

3. Install all dependencies:

```sh
npm install
```

4. Run locally:

```sh
npm run dev
```
